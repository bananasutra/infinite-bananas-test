# Run it locally

This workspace was authored without running Node commands in-editor.

## Install

```bash
npm install
```

## Dev

```bash
npm run dev
```

Then open the printed local URL.

## Share / Compare

- User A completes the 60 questions → gets a `Copy Link`.
- User B opens that link (it contains `?results=...`) → takes the same 60 questions.
- The app shows the comparison Weather Report + Bananometer.
